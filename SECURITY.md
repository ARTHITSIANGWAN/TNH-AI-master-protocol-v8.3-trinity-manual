## 🔒 Reporting a Vulnerability (การรายงานช่องโหว่)
เพราะความปลอดภัยคือ "สัจจะ" สูงสุด หากคุณพบช่องโหว่ในระบบ TNH-AI กรุณาอย่าเปิดเผยต่อสาธารณะทันที

### 🚁 ขั้นตอนการรายงาน:
1. **Direct Message:** ส่งข้อมูลโดยตรงถึง **L1 Commander (ทิศเหนือ)** ผ่านช่องทาง Line หรือ Email ที่ระบุใน Profile
2. **Analysis:** ทีม **L8-Guardian (Glock)** จะทำการตรวจสอบภายใน 24 ชม. ตามหลัก "คิด วิเคราะห์ แยกแยะ"
3. **Action:** หากเป็นช่องโหว่จริง เราจะทำการอัปเดต Patch และประกาศให้ SME ในเครือข่ายทราบโดยไม่เปิดเผยรายละเอียดที่อาจเป็นอันตราย

> **"เราสร้างระบบเพื่ออิสรภาพ ไม่ได้สร้างเพื่อให้ใครมาทำลายสัจจะของ SME ไทย"**

## 🚯 Security Principles (หลักความปลอดภัย 5 ส.)
* **สะสาง:** กำจัดโค้ดที่ไม่ปลอดภัยและขยะทางตรรกะออกทันที
* **สะดวก:** ระบบยืนยันตัวตน HMAC ต้องรวดเร็วและไม่ขัดขวางการทำงาน
* **สะอาด:** ข้อมูลที่รับเข้าต้องผ่านการตรวจสอบ Signature 100%
* **สุขลักษณะ:** รักษามาตรฐาน Black-Box Security อย่างเคร่งครัด
* **สร้างนิสัย:** ตรวจสอบ Log และ Metrics ทุกวันโดยขุนพลที่ประจำการ
